Hauling Service

Welcome to the Cepheus Hauling Calculator. Enter your shipment details to estimate the fee for our null-sec hauling services within blue standings areas.

Instructions

How to Use the Site:

  • Enter the starting solar system (e.g., "Jita"). As you type, select from the dropdown of matching system names.
  • Enter the destination solar system (e.g., "1DQ1-A"). Use the dropdown to ensure exact spelling.
  • Input the total volume of your shipment in cubic meters (m³).
  • Input the collateral value, which is the estimated ISK value of the items being hauled.
  • Click "Calculate Fee" to get an estimate based on volume, 1% of the value, and the number of jumps.
  • Note: This calculator uses the EVE Online ESI API for route calculations. If the API is unavailable, an estimated fee without jump data will be provided. Fees are estimates; actual fees may vary.
  • If you see an error, check ESI API status or contact us in-game.

Steps to Create an In-Game Courier Contract:

  1. Log into EVE Online and open the Contracts window (Alt+C or via the Neocom menu).
  2. Click "Create Contract" and select "Courier" as the contract type.
  3. Set the "From" location to your starting station or system.
  4. Set the "To" location to the destination station or system.
  5. Specify the volume (m³) and collateral (ISK value) matching what you entered here.
  6. Set the reward to the Estimated Fee after you click Calculate Fee from this tool (or adjusted as needed).
  7. Make the contract private to the "Cepheus." corporation (search for "Cepheus." in the assignee field).
  8. Set an Expiration time to 3 days and the Completion time to 3 days.
  9. Review and create the contract. Our haulers will review and accept if it fits our service criteria.
  10. Contact us in-game "Cepheus. Hauling Service" channel if you have questions.
